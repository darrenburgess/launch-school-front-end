CSS Selectors

A CSS selector defines which element the defined css properties apply to.

Basic selectors

Element Selector

The element type selector selects based on the html element such as `body`, `article`, `li`, etc

HTML
<p>How, now, brown, cow</p>
CSS
p { font-weight: bold; }
Result

How, now, brown, cow

Class Selector

HTML elements can have a class applied to them with this syntax with multiple classes delimited by spaces. In the CSS this element can be selected with the class selector.

HTML
<p class="bolded">How, now, brown, cow</p>
CSS
.bolded { font-weight: bold; color: blue; }
Result

How, now, brown, cow

ID Selector

HTML elements can also be assigned an id. Element ID's must be unique in the document. The hash symbol is used to denote an ID selector in the CSS

HTML
<p id="brown-cows">How, now, brown, cow</p>
CSS
#brown-cows { font-weight: bold; color: green; }
Result

How, now, brown, cow

Attribute Selector

Chooses nodes based on the value of an attribute assigned to the element.

There are several ways to target attributes. More examples can be found at MDN

HTML
<form action="#" method="get"> <input type="text" placeholder="Text type input"> </form> <p><span lang="en">this is a bold span</span></p> <p><span lang="ch sp">this a blue bold span</span></p>
CSS
/* select using the type attribute */ input[type="text"] { border: 2px solid blue; height: 30px; width: 150px; } /* all spans with a lang attribute are bold */ span[lang] { font-weight: bold; } /* all spans with a lang attribute = "sp" */ /* in a white space delimited list */ span[lang~="sp"] { color: blue }
Result

this is a bold span

this a blue bold span

Universal Selector

Selects all elements and can be chained to make it more specific. The universal selector can also be used to select all elements in the page. This is generally not a good idea.

HTML
<div class="universal"> <p><span>span 1 in the universal div</span></p> </div> <p><span>span 2 outside the div</span></p>
CSS
/* selects all elements in the div with class universal */ div.universal * { color: orange; }
Result

span 1 in the universal div

span 2 outside the div

Selector Combinators

Selectors can be combined together with various operators to achieve more precise element selection.

Adjacent Sibling (+)

Selects the element that immediately follows the former specified element.

General Sibling (~)

Selects all elements of a type that follow an element within the same parent, not necessarily immediately after.

Descendant ( )(a space)

Selects all elements that are child of the first element, not necessarily immediate children.

Direct Child (>)

Selects all of the elements that are directly a child of the first element

Grouping (,)

Selects all of the elements in the comma delimited list. This provides a way to apply the same styles to each selected element.

CSS
/* Adjacent Sibling: */ /* selects <p> that immediately */ /* follows <p class="some-class"> */ p.some-class + p { } /* General Sibling */ /* selects all <span> following a <p> in the same parent */ p ~ span { } /* Descendant */ /* selects all <p> elements inside of <div class="class"> */ div.class p { } /* Direct Child */ /* selects all <p> elements that are immediate */ /* children of <div class="class"> */ div.class > p { } /* Grounping */ /* selects all elements in the comma delimited list * h1, h2, h3, h4, .bolded { }

Pseudo Elements

Pseudo elements are abstractions of the dom tree that provide elements beyond the scope of normal html. They allow a way to style these abstracted elements independently. Syntax: selector::pseudo-element { property: value }

::after

Matches an inline (by default) virtual last child of the selected element. often used to add content via css, such as text, a background image or for the clear fix solution.

::before

An inline (by default) virtual first child of the selected element. Often used to add cosmetic content to the element such as a list-item graphic or an icon.

::first-letter

Provides ability to select the first letter of a line.

::first-line

Provides ability to select the first line of a paragraph.

::selection

Provides for a limited subset of properites to style user selected text. Such as the background-color and color.
